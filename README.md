## How to add games

If not present, create a folder called **games** next to the executable. Inside, create a folder for each game you want. The name of the folder will be the name of the game.

game_launcher.exe  
|_ games  
  |_ Game name  
    |_ game_executable.exe  
    |_ capsule.jpg
